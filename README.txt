Плагин работает с Adobe After Effects CS6

Каталоги:
AF 		- Необходимые фалы для работы плагина, (переместь всё в C:\Temp\AF\)
Build  	- Сам собранный плагин, необходимо скопировать в (c:\Program Files\Adobe\Common\Plug-ins\CS6\MediaCore\)
Fonts	- Шрифты, которые необходимо установить, что бы при открытии шаблона After Effects не спрашивал дополнительные сообщения
Sources	- Исходники проекта, для Visual Studio 2012, файл солюшена находится в каталоге CustomEffectMaker

Для работы плагина:
1) Все файлы из папки AF нужно перенести в папку C:\Temp\
т.е. полный путь к файлам должен быть C:\Temp\AF\
в этой папке должен быть файл logo.png, и каталог с шаблоном "Online Shop Promo"

2) Сам собранный плагин находится в каталоге "Build", и называется "MoveMaker.aex"
Его необходимо скопировать в каталог c:\Program Files\Adobe\Common\Plug-ins\CS6\MediaCore\

3) После чего запустить After Effects, в меню Edit появится первым пунктот Custom Move Maker

Описание работы:
После выбора пункта меню Custom Move Maker
- Плагин, выберет из папки c:\Temp\AF\Online Shop Promo\ проект шаблона, под названием Online Shop Promo.aep
- Пересохранит его с новм именем WorkingProject.aep
- Создаст новый фотаж в проекте в новой папке "Custom folder container" с изображением логотипа logo.png
который возмёт из папки c:\Temp\AF\logo.png
- Установит логотип его на слой "03 Import Your logo"
- Заменит цвет и текст на слое "01 Intro text"
- Так же заменит текст на слоях "02 Intro text" и "04 Intro text"
- Запусит процесс рендеринга клипа, результирующий файл будет находится в каталоге c:\Temp\AF\Working-output.avi 
- Так же, будет созданн файл WorkingProject.aep_Dump.txt с полням описанием содержимого проекта, по всем слоям

Примечание:
Настроки рендеренга, такие как размер и кодек, должны быть установленны в файле шаблона
так как сечас файл клипа сечас может достигать 14 Gb. из-за неустановленного кодека и размером 1920x1080
